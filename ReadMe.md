BallPlay Fantasy is a puzzle game, based on the "BallPlay" concept. More information will be added to this readme later.


### Repositories

[Codeberg](https://codeberg.org/PhantasarProductions/BallPlay_Fantasy)
Main repository. Whenever changes take place this is mostly the first repository to be updated. 
If you wanna clone or fork, I think Codeberg should be the place to be.

[GitHub](https://github.com/PhantasarProductions/BallPlay_Fantasy)
Secondary repository and basically just a clone of the Codeberg respository. Pushes may be a bit behind on the Codeberg repo. Issues should primary be posted on Codeberg. For the time being I will allow them on GitHub to be posted too, however, duplicates of issues posted on Codeberg will be closed on sight (even if the issue was posted on GitHub first), and some issues may even be copied to Codeberg and be handled from there (getting the Github version of the issue closed).

Please keep your pull requests on Codeberg and not on Github, thank you!

If you see this game on any other site, it could be either a backup site, or an unofficial clone/fork. I recommend to ignore those. (I'll mostly update this ReadMe file when another git repo site is added).


If you wanna follow what I'm doing live, well, perhaps you'll like my silly [live devlog](https://tricky1975.github.io/DevLogs/BallPlayFantasy/BallPlay_Fantasy_DevLog_Page_1.html), then?
